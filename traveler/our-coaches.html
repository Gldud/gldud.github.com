Front End Developer

Job Description

We are looking for amazing Javascript application developers who love crafting cutting-edge user experiences. The ideal candidate is someone who is comfortable delivering pixel perfect designs, as well as, sketching high-level features, and working with developers to implement features correctly.

Role Requirements

  • Minimum 3 years experience with JS, JQuery, HTML and CSS
  • Familiarity with grid/flexbox front-end frameworks such as Bootstrap, Foundation, SemanticUI, Jeet, Susy etc
  • Experience with Sass/Less, treeshaking and preprocessing for optimization.
  • Understanding of AJAX frameworks, REST and JSON
  • Desire to keep up-to-date on the latest front-end techniques and trends
  • Self-motivated, and can excel in a team-based environment
